# Google Play Store — Release notes (What's new)

Version **1.3.0**. One block per supported language, wrapped in its Play Store
locale tag. Character limit: **500 characters** per language (Play Console
"What's new" field).

<en-GB>
What's new in 1.3.0:
• MPEG-DASH companion playback is now available on iOS
• More accurate and stable synchronization between the TV and web players
• External immersive companion pages stay synchronized through a new connection bridge
• Stability fixes and refinements throughout
Thanks for using Universal MediaSync!
</en-GB>
