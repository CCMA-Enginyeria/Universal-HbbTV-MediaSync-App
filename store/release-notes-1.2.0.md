# Google Play Store — Release notes (What's new)

Version **1.2.0**. One block per supported language, wrapped in its Play Store
locale tag. Character limit: **500 characters** per language (Play Console
"What's new" field).

<en-GB>
What's new in 1.2.0:
• Immersive companion pages now open in your browser and stay in sync with the TV, so WebXR experiences work
• Richer previews for companion content, with title and thumbnail
• Optional camera support for the companion experience
• Stability fixes and refinements throughout
Thanks for using Universal MediaSync!
</en-GB>
