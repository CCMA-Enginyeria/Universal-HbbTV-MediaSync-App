# Google Play Store — Release notes (What's new)

Version **1.1.0**. One block per supported language, wrapped in its Play Store
locale tag. Character limit: **500 characters** per language (Play Console
"What's new" field).

<en-GB>
What's new in 1.1.0:
• Smoother, more accurate sync with predictive drift correction
• New companion web experience: follow synchronized content in your browser
• Clearer sync status so you always know when audio is locked to the TV
• Stability fixes and refinements throughout
Thanks for using Universal MediaSync!
</en-GB>
