# Google Play Store — Release notes (What's new)

Version **1.0.0** — first public release. One block per supported language,
wrapped in its Play Store locale tag. Character limit: **500 characters** per
language (Play Console "What's new" field).

<en-GB>
First release of Universal MediaSync!
• Discover HbbTV-enabled TVs on your Wi-Fi network (DIAL/SSDP)
• Private listening straight to your headphones
• Accessibility tracks: audio description and sign language
• Original version and alternative audio
• Frame-accurate DVB-CSS synchronization
• Playback keeps running in the background
• Available in Catalan, Spanish, Basque, English, German, Italian and French
</en-GB>
